# Just another MIDI player 

uses  [midifile](https://github.com/nfroidure/midifile) for parsing midi data.

# Motivation
* used by [werckmeister component](https://github.com/werckme/werckmeister-component) 



## Supported MIDI events
* Note On
* Note Off
* CC
  * experssion
  * panorama
* PC
* Pitch Bend
* Meta Events
  * Tempo
